Every Pro Diagnosis + Remedy Package starts with the same twenty-eight standard checks. But many audits we deliver include more. Sometimes a few more. Sometimes more than twenty more.

Clients occasionally ask if that means the bigger audits are "better" or "more thorough." It does not. It means the site had more to look at.

## Standard checks vs. additional diagnostics

The twenty-eight standard checks apply to every business website. Page title, meta description, schema markup, HTTPS, analytics, robots.txt, and so on. Those are universal. Every site has them or should.

The additional diagnostics, which we label B01 through B22, exist for things that aren't universal. Mobile layout issues that only show up when the design has certain elements. Sitemap errors that only matter if the site has more than ten pages. Multilingual hreflang tags that only apply to sites serving more than one country. Cookie consent banners that only become a check if the site has visitors from the EU or California. Video schema that only matters if the site has video.

We run an additional diagnostic when your site has the feature that makes the check relevant. We don't run it when the check would be empty noise.

## Why this is the honest way to price an audit

A scanner that checks every site for every possible signal would tell a brochure-site owner that they "failed" the e-commerce structured data check, the multilingual hreflang check, the video schema check, and the blog post canonical check. None of those findings would mean anything, because the site has no products, one language, no video, and no blog. But the list of "failures" would look long and intimidating.

That kind of inflated finding list is how some audit tools manufacture urgency. The longer the list, the bigger the perceived problem, the easier the upsell.

We do the opposite. Your audit contains the twenty-eight standard checks plus only the additional diagnostics that actually apply. If your site has fifteen things worth examining beyond the standard set, you get a fifteen-item B-section. If it has two, you get two. If it has none, the audit covers the twenty-eight, and that is the right answer.

## What this means for your report

When you open your Pro Diagnosis + Remedy Package and see, for example, "50 total findings (28 standard + 22 additional)," that is a real number. Every B-check in there was added because we saw something on your site that warranted including it. It is not padding.

You are paying for the checks that apply to your business, not for the appearance of being thorough.
